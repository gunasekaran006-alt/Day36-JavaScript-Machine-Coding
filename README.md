# Shopyfy - Simple E-commerce Cart Application 🛒

Shopyfy is a front-end web application that simulates a real-world shopping experience. It fetches dynamic product data from an external API and allows users to browse products, add them to a cart, and manage their selections.

## 🚀 Features
- **Dynamic Product Loading:** Fetches real-time product data (titles, prices, images) from `fakestoreapi.com`.
- **Shopping Cart Logic:** Users can add items to the cart, and the app handles duplicate items by increasing quantity.
- **Persistent Storage:** Uses `LocalStorage` so your cart items remain even after refreshing the page.
- **Cart Management:** View total price, remove specific items, or clear the entire cart.
- **Fully Responsive:** Built with **Bootstrap 5**, ensuring a smooth experience on mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **HTML5** & **CSS3**
- **JavaScript (ES6+)** - Fetch API, LocalStorage, Array Methods.
- **Bootstrap 5** - For UI components and styling.
- **FakeStoreAPI** - For mock product data.

## 📂 Project Structure
- `index.html` - The home page where products are displayed.
- `cart.html` - The checkout page to view and manage selected items.
